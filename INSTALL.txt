# Delhi Road Data Dashboard - インストール手順

## 必要な環境
- Python 3.x がインストールされていること
- インターネット接続（地図データ取得のため）

## 配布ファイル
このパッケージには以下が含まれています：
- index.html（ダッシュボード本体）
- geojsonフォルダ（データファイル）
- start_server.bat（簡単起動用）
- README.md（詳細説明）

## 簡単な使用方法
1. フォルダ全体を任意の場所に展開
2. `start_server.bat` をダブルクリック
3. ブラウザでダッシュボードが開きます

## 政府プロジェクト配布時の注意
- フォルダ構造を維持してください
- Python環境が必要です
- データファイルは約100MBあります
- オフライン使用時は地図表示が制限される場合があります
